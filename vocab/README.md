# Vocab - Словарная база данных

Коллекция словарей для английского и русского языков.

## Структура проекта

```
vocab/
├── en/
│   └── words.txt    # Английские слова (466,550 слов)
└── ru/
    └── words.txt    # Русские слова (1,532,628 слов)
```

## Описание

Проект содержит два больших словаря:

- **Английский словарь**: 466,550 слов в файле `en/words.txt`
- **Русский словарь**: 1,532,628 слов в файле `ru/words.txt`

Всего в базе содержится **1,999,178 слов**.

## Формат данных

Слова хранятся в текстовых файлах, по одному слову на строку. Файлы кодированы в UTF-8.

### Примеры слов

**Английские слова:**
```
abatage
abattage
abedge
abegge
abjudge
```

**Русские слова:**
```
актрисо
асессо
бассо
белесо
белобрысо
```

Эти словари могут использоваться для:
